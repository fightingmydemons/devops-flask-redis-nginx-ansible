# Автоматизированный стек деплоя (Nginx + Flask + Redis) через Ansible

Проект демонстрирует развертывание безопасной микросервисной архитектуры на удаленном сервере под управлением Linux (Rocky/RHEL) с использованием лучших практик DevOps.

## Архитектура проекта
* **Nginx** (Порт 80) — выступает в роли Reverse Proxy, защищая бэкенд и распределяя трафик.
* **Flask (Python)** — изолированное приложение бэкенда, обрабатывающее бизнес-логику (счетчик посещений).
* **Redis** — in-memory база данных для быстрого хранения состояния счетчика.

Вся инфраструктура полностью изолирована внутри Docker-сетей. Доступ извне открыт только к Nginx (порт 80).

## Технологический стек
* **OS:** Arch Linux (CachyOS) на хосте, Rocky Linux на сервере.
* **Контейнеризация:** Docker, Docker Compose.
* **Оркестрация / Конфигурация:** Ansible.
* **Веб-сервер:** Nginx (Reverse Proxy).
* **Скрипты:** Bash, YAML.

## Как запустить
1. Настроить беспарольный доступ по SSH к вашему серверу.
2. Прописать IP сервера в файл `hosts.ini`.
3. Запустить плейбук автоматического деплоя:
   ```bash
   ansible-playbook -i hosts.ini playbook.yml -K
